Build a Kubernetes Cluster Locally with Minikube

Objective: Deploy and manage applications in Kubernetes

Tools Used

  • Minikube
  • kubectl
  • Docker

Deliverables

  • YAML configuration files (deployment.yaml, service.yaml)
  • Screenshots of running Pods and Services

Steps / Hints

  1. Install Minikube & start the cluster: 
    minikube start
  2. Create deployment.yaml for an app and apply it: 
    kubectl apply -f deployment.yaml
  3. Expose app using service.yaml: 
    kubectl apply -f service.yaml
  4. Verify running pods: 
    kubectl get pods
  5. Check services: 
    kubectl get svc
  6. Scale deployments: 
    kubectl scale deployment my-app --replicas=3
  7. Describe resources and check logs: 
    kubectl describe pod my-app-pod

Conclusion

By following these steps, you can set up a local Kubernetes cluster using Minikube, deploy containerized applications, expose them as services, and manage scaling and monitoring using kubectl.
